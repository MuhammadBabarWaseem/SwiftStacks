# Its a small deployment web project

## Tech Stack Used

### For Backend
<ul>
  <li>TypeScript</li>
  <li>Express</li>
  <li>AWS SDK</li>
  <li>Cloudflare</li>
  <li>Redis For Queuing</li>
</ul>

### For Backend
<ul>
  <li>React.js</li>
  <li>ShadCN</li>
  <li>Tailwind CSS</li>
</ul>

### How to run this
<ul>
  <li>cd folder</li>
  <li>First create build by: tsc -b</li>
  <li>Then run by: node dist/index.js</li>
</ul>
